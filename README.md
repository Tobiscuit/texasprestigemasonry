# Texas Prestige Masonry

Premium masonry services platform built with Next.js 16, Payload CMS, and an autonomous SEO engine.

## Services
- Outdoor Kitchens
- Custom Pavers
- Chimneys
- Brick & Block
- Fire Pits
- Stone Repairs

## Stack
- **Framework**: Next.js 16 (App Router)
- **CMS**: Payload CMS 3.x (Postgres)
- **Styling**: Tailwind CSS v4
- **i18n**: next-intl (EN, ES, VI)
- **SEO**: Autonomous SEO Engine (Cloudflare Worker)
- **AI**: Gemini-powered content generation dashboard

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and configure your database, S3, and API keys.
